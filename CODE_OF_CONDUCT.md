
## TLFloat Code of Conduct

* Abolish the idea that open source or free software should be used
  free of charge without contributing anything.

* Promote the idea that when open source or free software is used to
  make a profit, a portion of that profit should be returned to the
  corresponding project.

* Promote the idea that if an organization uses open source and free
  software, it should also contribute to open source and free software
  projects.

* Promote the idea that members of an organization who contribute to
  open source and free software projects should be given sufficient
  recognition as part of their work.
